# AdsPowerManager
Библиотека для работы с антидетект браузером AdsPower.
-Создание браузера
-Закрытие браузера (профиля)
-Создание профиля
-Удаление профиля
-Получение профилей
-Получение групп

## Установка

Для использования этой библиотеки, убедитесь, что у вас установлены следующие зависимости:

- Newtonsoft.Json
- OpenQA.Selenium
- OpenQA.Selenium.Chrome
